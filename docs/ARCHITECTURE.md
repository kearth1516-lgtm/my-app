
# アーキテクチャ設計書

## 全体構成

```
┌─────────────────────┐
│   React + TypeScript │ (フロントエンド)
│   モバイル対応UI     │
└──────────┬──────────┘
           │ HTTP/REST API
┌──────────▼──────────┐
│  FastAPI (Python)   │ (バックエンド)
│  ビジネスロジック   │
└──────────┬──────────┘
           │
┌──────────▼──────────┐
│  Azure Cosmos DB    │ (データベース)
│  NoSQL Document DB  │
└─────────────────────┘
```

## 機能モジュール構成

### Phase 1（MVP）
- ホーム画面（推し写真表示）
- タイマー機能（記録機能付き）
- レシピ管理（基本CRUD）

### Phase 2
- ファッション管理
- 統計ダッシュボード
- スケジュール同期

### Phase 3
- AI提案機能（RAG）
- 通知機能

## 技術スタック

| レイヤー | 技術 | 理由 |
|---------|------|------|
| フロントエンド | React + TypeScript | 型安全、モバイル対応 |
| バックエンド | FastAPI | 高速、Python、自動ドキュメント |
| DB | Azure Cosmos DB | スケーラブル、無料枠あり |
| ホスティング | Azure App Service | 無料枠対応、Python対応 |
| 管理 | GitHub | ドキュメント + リポジトリ |
